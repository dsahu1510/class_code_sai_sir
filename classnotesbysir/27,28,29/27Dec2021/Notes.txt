Colors:

CSS provides around 140 different colors to use for the web pages.

we need to use color property to attach some color to a html element.


<html>
    <head>

         <style>
         
            h1{
              color : #6BC8D0
            }

         </style>

    </head>

     <body>

        <center> <h1>header</h1> </center>

     </body>
</html>
